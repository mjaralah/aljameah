## الخطة

### 1) تحسين صفحة `/admin/support-page` الحالية (إدارة المحتوى)

ملف وحيد: `src/pages/admin/AdminSupportPage.tsx`

- إضافة `dir="rtl"` للحاوية الرئيسية وضبط المحاذاة لليمين في كل البطاقات والحقول والأزرار.
- استبدال شريط التبويبات الحالي بـ **Tabs بأيقونات ملونة + Badges للعدّ**:
  - الإعدادات العامة → أيقونة `Settings` (لون أساسي).
  - التصنيفات → أيقونة `LayoutGrid` (أزرق) + Badge بعدد التصنيفات.
  - الأسئلة الشائعة → أيقونة `HelpCircle` (بنفسجي) + Badge بعدد الأسئلة.
  - الروابط السريعة → أيقونة `Link2` (برتقالي) + Badge بعدد الروابط.
- تكبير حجم التبويبات (`h-12`) مع `data-[state=active]` لتمييز التبويب النشط بخلفية ملونة وحدود سفلية، وحالة hover واضحة.
- ترقية البطاقات الداخلية (Cards) بظلال خفيفة، فواصل، وعناوين أقسام أوضح.
- زر «معاينة الموقع» يبقى أعلى يسار.

### 2) صفحة جديدة: `/admin/help` — مركز دعم المدير

صفحة احترافية للمدير غير التقني، **شبيهة بصفحة `/support` العامة** لكن بمحتوى من جدول `help_articles` الموجود.

**التركيبة (نفس بنية صفحة `/support`):**

```text
┌───────────────────────────────────────────────────┐
│  Hero بحث (Fuse.js على help_articles)             │
│  «كيف يمكننا مساعدتك في إدارة الموقع؟»             │
├───────────────────────────────────────────────────┤
│  شبكة تصنيفات بأيقونات (من categories الموجودة)    │
│  [الإعدادات] [المحتوى] [الأخبار] [النماذج] ...      │
├───────────────────────────────────────────────────┤
│  مقالات/أسئلة شائعة (Accordion)                    │
│   • كل إجابة تدعم Markdown/HTML آمن                │
│   • أزرار «اذهب إلى الصفحة» (action_url من المقال)│
│   • تضمين فيديو/صورة (إذا توفر media_url)         │
├───────────────────────────────────────────────────┤
│  نموذج تواصل مع الدعم التقني │ روابط سريعة         │
└───────────────────────────────────────────────────┘
```

**الميزات:**
- بحث لحظي بـ Fuse.js مع التطبيع العربي والمرادفات (نفس `arabicNormalize.ts`).
- النقر على تصنيف يفلتر القائمة + يمرّر للأسفل.
- **أزرار إجراءات مباشرة** بجانب كل إجابة: إذا كان للمقال `action_url` (مثل `/admin/settings`) يظهر زر «اذهب الآن».
- **تضمين فيديو/صورة**: إذا كان للمقال `media_url` و`media_type` يُعرض داخل الإجابة (iframe لـ YouTube/Vimeo أو `<img>`).
- **نموذج تواصل مع الدعم التقني**: يُرسل إلى `contact_messages` مع `purpose = "دعم تقني - إدارة"` وملء البريد من حساب المدير تلقائياً.
- **روابط سريعة لأدلة PDF/فيديو**: يقرأ من `support_quick_links` (مشترك مع الصفحة العامة) — مع شارة «روابط الإدارة فقط» اختيارية لاحقاً.

### 3) قاعدة البيانات

توسعة جدول `help_articles` بحقول جديدة (اختيارية، لا تكسر البيانات الحالية):

| العمود | النوع | الوصف |
|---|---|---|
| `action_url` | text | رابط داخلي لزر إجراء مباشر |
| `action_label` | text | نص الزر (افتراضي «اذهب الآن») |
| `media_url` | text | رابط فيديو/صورة |
| `media_type` | text | `image` / `youtube` / `vimeo` |

تحديث policies الموجودة لا يحتاج تعديل (الحقول الجديدة تتبع نفس RLS).

### 4) التنقل والربط

- إضافة عنصر في الشريط الجانبي للوحة التحكم: **«مركز المساعدة»** يفتح `/admin/help` (يبقى عنصر «إدارة مقالات المساعدة» الحالي `/admin/help-center` لإدارة المحتوى).
- زر داخل المساعد العائم: «افتح مركز المساعدة الكامل» يفتح `/admin/help` في تبويب جديد.
