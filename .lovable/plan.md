# نقل بطاقة معلومات الدورة إلى يمين بطاقات الأعضاء

## الهدف
تحسين التسلسل البصري في قسم "أعضاء مجلس الإدارة" بنقل بطاقة (مدة الدورة، تاريخ الانتهاء هجري/ميلادي، قرار التشكيل) من شريط علوي إلى عمود جانبي يميني بمحاذاة شبكة بطاقات الأعضاء.

## التصميم المعتمد

```text
+-------------------------------------------------------------+
|  [بطاقات الأعضاء — 3/4 العرض]      |  [معلومات الدورة 1/4]  |
|  +-------+ +-------+ +-------+      |  مدة الدورة            |
|  | عضو 1 | | عضو 2 | | عضو 3 |      |  تاريخ الانتهاء        |
|  +-------+ +-------+ +-------+      |  هجري / ميلادي         |
|  +-------+ +-------+                |  ---------------       |
|  | عضو 4 | | عضو 5 |                |  [تحميل قرار التشكيل]  |
|  +-------+ +-------+                |                        |
+-------------------------------------------------------------+
```

- **الشاشات الكبيرة (lg+):** عمودان `grid-cols-4`؛ بطاقات الأعضاء `col-span-3` على اليسار، بطاقة معلومات الدورة `col-span-1` على اليمين (في RTL يظهر العمود تلقائياً يميناً).
- **الشاشات الصغيرة (md وأقل):** بطاقة معلومات الدورة تظهر **أعلى** بطاقات الأعضاء بعرض كامل.
- **الالتصاق:** لا — تمرير طبيعي مع المحتوى.
- داخل بطاقة المعلومات: شبكة عمود واحد بثلاث خانات منفصلة (المدة، تاريخ الانتهاء، زر تحميل القرار) بدلاً من ثلاث بطاقات أفقية.

## الملف الوحيد المتأثر
- `src/pages/About.tsx` — قسم Board فقط:
  - استبدال التخطيط الحالي (شريط علوي + شبكة أعضاء كاملة العرض) بـ `grid lg:grid-cols-4 gap-6`.
  - تجميع بطاقات الأعضاء داخل `lg:col-span-3` مع شبكة داخلية `md:grid-cols-2`.
  - بطاقة معلومات الدورة في `lg:col-span-1` بترتيب `order-first lg:order-last` لضمان ظهورها أعلى البطاقات على الجوال ويميناً على الديسكتوب.
  - إعادة هيكلة محتوى البطاقة الجانبية: 3 صفوف عمودية بفواصل خفيفة (`divide-y border-border`).

## بدون تغييرات
- لا تعديل على قاعدة البيانات.
- لا تعديل على لوحة التحكم.
- نفس البيانات والـ hooks (`useBoardSettings`, `useBoardMembers`).
- نفس هوية الموقع (ألوان teal/gold، `bg-card`, `rounded-xl`).
