## إعادة تصميم قسم "أعضاء مجلس الإدارة"

### 1. تغييرات قاعدة البيانات

**أ) جدول `board_members` — إضافة عمود:**
- `term_duration` (text) — مدة الدورة لهذا العضو (مثل "4 سنوات")

**ب) جدول جديد `board_settings` (سجل واحد) لإعدادات القسم العامة:**
- `term_duration_label` (text) — العنوان العام (مثل "4 سنوات")
- `term_end_hijri` (text) — تاريخ انتهاء هجري (مثل "1429-06-11")
- `term_end_gregorian` (date) — تاريخ انتهاء ميلادي
- `show_hijri` (boolean, default true) — إظهار/إخفاء الهجري
- `show_gregorian` (boolean, default true) — إظهار/إخفاء الميلادي
- `formation_decree_url` (text) — رابط ملف PDF لقرار التشكيل
- `formation_decree_name` (text) — اسم الملف المعروض
- `intro_text` (text) — النص التعريفي للقسم (بدلاً من النص الثابت الحالي)

سياسات RLS: قراءة عامة + كتابة للموظفين (مطابقة لباقي جداول المحتوى).

### 2. الواجهة العامة — `src/pages/About.tsx` قسم `#board`

تخطيط جديد بهوية الموقع الحالية (نفس الألوان: primary teal + gold accent، نفس البطاقات `bg-card border-border rounded-xl`):

```text
┌─────────────────────────────────────────────────────────┐
│ [أيقونة] أعضاء مجلس الإدارة                              │
│ النص التعريفي                                            │
├──────────────────────────────┬──────────────────────────┤
│  بطاقات الأعضاء (شبكة 2-3)    │  لوحة معلومات جانبية:    │
│  ┌─────────┐  ┌─────────┐   │  • مدة الدورة (4 سنوات)  │
│  │ صورة    │  │ صورة    │   │  • تاريخ الانتهاء:       │
│  │ الاسم   │  │ الاسم   │   │    - هجري                │
│  │ المنصب  │  │ المنصب  │   │    - ميلادي              │
│  │ النبذة  │  │ النبذة  │   │  • قرار تشكيل المجلس     │
│  │ ───     │  │ ───     │   │    [تحميل الملف المرفق]  │
│  │⏱ 4سنوات│  │⏱ 4سنوات│   │                          │
│  └─────────┘  └─────────┘   │                          │
└──────────────────────────────┴──────────────────────────┘
```

- البطاقة: تحتفظ بالصورة + الاسم + Badge للمنصب + النبذة، ويُضاف فاصل ثم سطر `⏱ مدة الدورة: {term_duration}` (يظهر فقط إن وُجد).
- اللوحة الجانبية (sticky على الشاشات الكبيرة، تظهر أعلى البطاقات على الموبايل): 3 خانات منفصلة لمدة الدورة، تاريخ الانتهاء (هجري/ميلادي حسب الإعدادات)، وزر تحميل قرار التشكيل (`Button` بأيقونة `Download`، يظهر فقط إن رُفع الملف).

### 3. لوحة التحكم

**أ) `src/pages/admin/AdminBoardPage.tsx`:**
- إضافة حقل `term_duration` في نموذج العضو (حقل نصي اختياري).

**ب) صفحة جديدة `src/pages/admin/AdminBoardSettingsPage.tsx` (أو tab داخل صفحة المجلس):**
- نص تعريفي القسم
- مدة الدورة (نص)
- تاريخ الانتهاء هجري (نص) + Switch لإظهاره
- تاريخ الانتهاء ميلادي (date) + Switch لإظهاره
- رفع/استبدال ملف قرار التشكيل عبر `MediaUpload` على bucket `documents`

أفضّل إضافة **تبويبين** داخل `/admin/board`: "الأعضاء" و"إعدادات المجلس" للحفاظ على رابط واحد في القائمة.

### 4. الملفات المتأثرة

- `supabase/migrations/<timestamp>_board_term_and_settings.sql` — إنشاء العمود + الجدول + RLS + سجل افتراضي.
- `src/integrations/supabase/types.ts` — يُحدَّث تلقائياً.
- `src/pages/About.tsx` — استبدال block `#board` بالتخطيط الجديد، جلب `board_settings`.
- `src/pages/admin/AdminBoardPage.tsx` — إضافة `term_duration` + هيكلة Tabs.
- `src/pages/admin/AdminBoardSettingsPage.tsx` — جديد (نموذج إعدادات القسم).

### تفاصيل تقنية

- يُجلب `board_settings` بصف واحد عبر `.maybeSingle()` ويُمرَّر للقسم.
- زر التحميل يستخدم `<a href={formation_decree_url} target="_blank" download>` مع أيقونة `Download` من lucide.
- التواريخ تظهر داخل بطاقة بأيقونة `CalendarCheck`؛ مدة الدورة بأيقونة `Clock`؛ القرار بأيقونة `FileDown`.
- لا تغيير في باقي أقسام `/about` ولا في باقي الموقع.
